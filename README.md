### Prerequisites

In order to run this project you need:

Web Browser (Chrome recommended)<br>
Code editor (VS recommended)<br>
GitHub account<br>

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone "https://github.com/danielochuba/Techdice-Web-backend.git"
```
### Install

Go to your /house_booking_backend folder:

```sh
  cd /Techdice-Web-backend
  bundle install
  rails s
```
- SETUP [Front-end](https://github.com/Techdiceitsolutions/Techdice-Website-Frontend) 

```sh
  cd /duo-cafe-frontend
  npm install
  npm run start
```	

### Usage

```sh
  npm start
```
